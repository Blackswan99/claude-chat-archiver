Claude Chat Archiver

Welcome to Claude Chat Archiver

This extension uploads your selected Claude.ai chats (messages and attachments) to a GitHub repository that you configure.

Data is transmitted directly between your browser, claude.ai, and api.github.com — never to any third party, analytics, or the extension author.

Nothing is uploaded automatically. You explicitly pick which chats to archive and trigger each sync yourself. Your GitHub token is stored locally in your browser.
